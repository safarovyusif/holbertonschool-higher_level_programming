Test cases for MyList class that inherits from list.

The primary test confirms the `print_sorted` method works and does not modify the original list.

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

Test case with a list initialized directly.

>>> new_list = MyList([10, -5, 0, 7])
>>> new_list.print_sorted()
[-5, 0, 7, 10]
>>> new_list
[10, -5, 0, 7]
